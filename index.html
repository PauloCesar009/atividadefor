Responda as perguntas abaixo:

1-Na tag input com atributo type="text", para que serve o atributo "placeholder"?

R= O atributo "placeholder" em uma tag input com o tipo "text" é usado para fornecer um texto de exemplo ou uma sugestão sobre o tipo de entrada esperado no campo de entrada.

2-Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

R= O atributo "checked" em uma tag input com os tipos "radio" ou "checkbox" serve para especificar que o botão de opção (no caso do tipo "radio") ou a caixa de seleção (no caso do tipo "checkbox") deve ser inicialmente marcado quando a página é carregada. Isso significa que, quando o atributo "checked" está presente, o botão de opção estará selecionado ou a caixa de seleção estará marcada automaticamente sem que o usuário precise interagir com ela.

3- Na tag input para que serve o atributo "value"?

R= O atributo "value" em uma tag input é usado para especificar o valor inicial do campo de entrada.

4-Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

R=Para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez, você deve usar o atributo "name" com o mesmo valor para todos os botões de rádio no mesmo grupo.

5-Para que serve a tag label? Para que serve o atributo "for" nessa tag?

R= o atributo "for" da tag label tem o valor "nome", correspondendo ao atributo "id" do campo de entrada associado. Isso permite que os usuários cliquem no rótulo "Nome:" para focar no campo de entrada de texto.

6-Na tag option para que serve o atributo "selected"?

R= O atributo "selected" na tag option é usado para especificar que uma opção dentro de um elemento select deve ser selecionada por padrão quando o menu suspenso é exibido pela primeira vez.

7- Nas tags de formulário para que serve o atributo "required"?

R=O atributo "required" em tags de formulário é usado para indicar que um campo é obrigatório, ou seja, que o usuário deve preencher esse campo antes de enviar o formulário.

8- Para que serve a tag textarea?

R=A tag textarea é usada em HTML para criar uma área de texto multi-linhas em um formulário web.

9-Na tag form para que serve o atributo "action"?

R=O atributo "action" na tag form é usado para especificar para onde os dados do formulário serão enviados quando o formulário é submetido. Ele contém um URL que representa o destino do formulário.

11-Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

R=O atributo "method" na tag form não tem relação com a visibilidade da senha na barra de endereços do navegador. A visibilidade da senha na barra de endereços do navegador está relacionada ao método de envio dos dados do formulário (GET ou POST) e ao tipo de campo de entrada usado para a senha.

12-Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

R= Para formulários que serão usados para fazer upload de arquivos, o valor do atributo "method" na tag form deve ser definido como "POST". Isso porque o método "POST" permite enviar dados, incluindo arquivos, no corpo da solicitação HTTP de forma segura.

13-O que as tags fieldset e legend produze?

R=As tags fieldset e legend são usadas em conjunto para criar grupos de controles de formulário HTML e fornecer uma legenda descritiva para esses grupos.
